# Contributing to Beast Mode

We love contributions! Please follow these steps to keep the beast tame.

## Release Hygiene

### 1. Update Changelog
Add your changes to `CHANGELOG.md` under `[Unreleased]`.

### 2. Verify Integrity
Run the validation script before submitting:
```bash
./scripts/verify-integrity.sh
```

### 3. Create Pull Request
- Use standard PR titles (feat:, fix:, docs:)
- Ensure all artifacts are documented

## Workflow Guidelines

- **Micro-Fix:** Keep it light. No heavy PM artifacts.
- **Enterprise:** Verify strict gates.
- **Templates:** Keep them copy-paste friendly.
